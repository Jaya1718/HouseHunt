# Househunt - MERN Stack Property Rental App

A full-stack property rental application built with:

- **MongoDB**
- **Express.js**
- **React.js** (Vite)
- **Node.js**

## Features

- User Authentication (JWT)
- Property Listings
- Booking System
- Admin & User Dashboards
- Responsive Design

## Deployment

- Backend: Render
- Frontend: Vercel
